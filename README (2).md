# Sales Performance Dashboard | Tableau

## Project Overview

This project analyzes a significant decline in sales from **$1.77M to $910K** using **Excel** for data preparation and **Tableau** for visualization. The dashboard provides a comprehensive view of key business metrics and highlights the factors contributing to the decline.

## Tools Used
- Excel (Data Cleaning & Preparation)
- Tableau (Data Visualization & Dashboard Development)

## Dashboard Highlights
- Total Sales, Profit, and Orders
- Year-over-Year Performance Comparison
- Sales by Product Category
- Regional Sales Distribution
- Monthly Sales Trends

## Key Insight
The analysis revealed that the sales decline was concentrated in specific product categories and regions rather than affecting the entire business.
